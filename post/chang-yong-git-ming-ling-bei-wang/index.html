常用git命令备忘

2020-12-11 1 min read

下载项目及代码历史

git clone [url]

查看项目本地分支

git branch

查看项目远程分支

git branch -r

查看项目所有分支（本地&远程）

git brnach -a

新建本地分支（但不切换）

git branch [本地]

删除本地分支

git branch -d [本地]

显示变更情况

git status

切换到一个本地分支

git checkout [本地]

新建一个本地分支并切换到这个分支

git checkout -b [本地]

新建一个本地分支，同步远程分支并切换到这个分支

git checkout -b [本地] [远程]

添加当前目录所有文件到暂存区

git add .

提交暂存区文件到仓库区

git commit -m ""

推送到远程仓库

git push

拉取最新代码

git pull
